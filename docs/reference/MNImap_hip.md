# Hippocampal surface in MNI space

A matrix with 14524 columns corresponding to the hippocampal template
vertices and 3 rows corresponding to each vertex's X,Y,Z coordinates in
MNI space

## Usage

``` r
MNImap_hip
```

## Format

### `MNImap_hip`

A 3x14524 matrix object

- coordinates:

  14524 rows (vertices), 3 columns (X,Y,Z coordinates)
