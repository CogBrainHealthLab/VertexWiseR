# List of edges for the hippocampal template

A Nx2 matrix object listing each vertex of the hippocampal template and
the vertices adjacent to it (making an edge together).

## Usage

``` r
edgelist_hip
```

## Format

### `edgelist_hip`

- Nx2 matrix object:

  Matrix with two columns and N rows corresponding to the unique edges
  in the fsaverage5 surface
