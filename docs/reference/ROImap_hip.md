# Atlas parcellations of the hippocampus (CITI168)

A list containing 1) a matrix listing CITI168 vertices and each
parcellation number they correspond to, and 2) a data frame listing the
hippocampal atlas labels.

## Usage

``` r
ROImap_hip
```

## Format

### `ROImap_hip`

A list object with two data frame objects: ()

- vertices:

  array of 14524 numeric vectors (vertices)

- atlases:

  data frame with 10 rows listing names of left and right hippocampal
  subfields
